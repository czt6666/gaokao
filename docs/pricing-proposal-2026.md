## 一、背景

老板提出新增 ¥30/年 会员 + 保留 ¥1.99 单篇解锁的双层方案。基于竞品调研重新审视后，建议不做 ¥30/年，而是把整个定价体系上调一个段位。

核心判断：**¥1.99 不是颠覆性定价，是自我贬值。** 家长看到"这么便宜"第一反应不是"赚到了"，是"数据是不是假的"——价格本身在杀转化率，不是在拉转化率。

## 三、建议方案

### 主推：单次完整报告 ¥39

- 锚点：掌上高考 ¥98 的 40%——"懂行的便宜，不是地摊货"
- 心理价位话术："比一杯星巴克贵一点 = 一份认真的填报参考"
- 绑定 (省份 + 位次 + 选科)，**永久有效**
- 预期承担 ~70% 流水

### 次推：填报季会员 ¥99 / 季

- 时间窗：购买日 → **当年 7 月 31 日**（不是 365 天）
- 权益：任意省份/位次/选科组合无限查 + 后续志愿表调整 + 优先客服
- 心智话术："家里查 3 次以上就回本（¥39 × 3 = ¥117 > ¥99）"
- 占 ~20% 流水，但 **LTV 与数据沉淀价值最高**

### 钓鱼层（可选）：¥9.9 试看

- 仅解锁前 3 所冷门校
- 漏斗：免费 → ¥9.9 试看 → ¥39 完整 / ¥99 季卡
- 让 ¥39 看起来"理所当然该买全套"
- 本身不指望赚钱，是漏斗工具

## 四、不选项的理由

### 为什么不做 ¥30/年

| 问题 | 解释 |
|---|---|
| 品类周期错配 | 高考志愿是单次决策，孩子明年高考时家长就走了，"年卡"在这个品类**没有续费基础** |
| 价格信号继续偏低 | ¥30 ÷ 365 ≈ 8 分/天，和 ¥1.99 一个心智段位，家长照样犯嘀咕 |
| 重复使用真窗口是 4–7 月 | 这个生意的真实形态是"高考季限定卡"，不是 365 天年卡 |

### 为什么不直接对标 ¥360 VIP

水卢目前**没有支撑 ¥360 价位的信任要件**：

- 客服团队（无）
- 真人专家（无）
- 内容矩阵（完美志愿 2000 条视频 vs 我们 0 条）
- 政府背书（掌上高考有教育部合作 vs 我们无）
- 品牌认知（夸克百度日活亿级 vs 我们品牌为零）

硬定 ¥360 转化率会从 1% 量级掉到 0.1% 以下，**总收入反而比 ¥39 低**。

## 五、风险与应对

| 风险 | 应对 |
|---|---|
| ¥39 比 ¥1.99 贵 20×，"随手买"用户被劝退 | 详情页加足信任锚：用户证言、对比表、机构 ¥3000 反衬、9 年数据 + 算法详情前置 |
| 家长不买 | 上 ¥9.9 试看作钓饵，让 ¥39 显得理所当然 |
| 品牌认知不够撑 ¥39 | 把"袁希团队 / 数据来源 / 算法依据"放到付费页显眼处 |
| 老用户怎么办 | 已付 ¥1.99 的订单不动、不追溯，新订单走新价 |

**敏感性测算**：¥39 比 ¥1.99 贵 20 倍。就算转化率腰斩到 0.3×（极端情况），营收仍涨 ~6×。只要转化率不掉到 0.05× 以下都是赚的。
